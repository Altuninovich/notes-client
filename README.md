# NOTES
Индивидуальные заметки каждого зарегистрированного пользователя с динамическим отображением времени с момента создания заметки. 

* Pезультат можно посмотреть тут: https://altuninovich.github.io/notes-client/


## SETUP
```sh
git clone
```
```sh
$ npm install
```
## RUN

```sh
$ npm start
```
